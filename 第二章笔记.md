# 第二章笔记

1. 变量名不能以数字打头

2. 字符串大小写：

   - 首字母大写：`var.title()`

   - 全 部 大 写：`var.upper()`

   - 全 部 小 写：`var.lower()`

3. 换行：`\t`

4. 制表符(tab)：`\n`

   **注意斜杠方向，同时 \t 与  \n 都是字符，应用“ ”括起**

5. 删除字符串中的空白：

   - 删除字符串两端空白：`var.strip()`
   - 删除字符串前端空白：`var.lstrip()`
   - 删除字符串后端空白：`var.rstrip()`

6. 将字符串 23 让电脑识别为数值23，而不是字符2和3：

   ```
   age=23

   str(age)
   ```

7. 支持中文`#coding=utf-8`   或者：`#coding=gbk`
